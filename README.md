# HR Analytics Dashboard

## 📌 Project Overview

This project is an interactive HR Analytics Dashboard created using Power BI. It helps analyze employee data to identify attrition trends, compare departments and job roles, and provide insights for better HR decision-making.

## 🎯 Objectives

- Analyze employee information
- Identify employee attrition patterns
- Compare attrition across departments and job roles
- Monitor salary and experience trends
- Build key HR performance indicators (KPIs)

## 📊 Dashboard Features

- Total Employees
- Attrition Count
- Attrition Rate
- Average Age
- Average Salary
- Average Years at Company
- Attrition by Gender
- Attrition by Age Group
- Attrition by Education
- Attrition by Salary
- Attrition by Job Role
- Attrition by Years at Company
- Department Filter (Slicer)

## 🛠 Tools Used

- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)

## 📂 Dataset

HR Analytics Dataset containing employee details such as:
- Age
- Gender
- Department
- Job Role
- Salary
- Years at Company
- Attrition
- Education
- Experience

## 📈 Key Insights

- Calculated employee attrition rate
- Compared attrition across departments
- Identified job roles with the highest attrition
- Analyzed salary and experience trends
- Built an interactive dashboard with filters for better analysis

- HR Data Analysis
